### 프로젝트 소개
시간과 공간을 크게 제약받지 않고 전문가 없이 간단하게 진행하는 색각이상 검사 개발

### 프로젝트 인원
3인 개발<br>
전체적인 개발을 담당

### 주요 기능
- 색각 이상 테스트 이미지 제공
- 사용자 입력 기반 결과 확인
- 간단한 UI를 통한 테스트 진행
- 입력값 저장 후 답안 비교
